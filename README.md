# 🌍 Bilingual Support (Arabic / English)

The project supports both:

* 🇸🇦 Arabic (RTL)
* 🇺🇸 English (LTR)

Using CSS directional selectors:

```css
:dir(rtl)
:dir(ltr)
```

No JavaScript is required for language handling.

# 📤 GitHub Deployment Guide

## Initialize Git Repository

```bash
git init
git add .
git commit -m "feat: professional bilingual portfolio"
```

---

## Connect to GitHub

```bash
git remote add origin https://github.com/YOUR_USERNAME/web-portfolio.git
git branch -M main
git push -u origin main
```

---

## Enable GitHub Pages

### Steps:

1. Open Repository Settings
2. Navigate to **Pages**
3. Select:

   * Branch: `main`
   * Folder: `/root`
4. Save Changes

Your website will become live at:

```bash
https://your-username.github.io/web-portfolio/
```
